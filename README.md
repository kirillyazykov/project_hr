# Онлайн-каталог сотрудников

## Описание
Веб-приложение на основе Flask и SQLAlchemy, отображающее информацию о сотрудниках из базы данных и предоставляющее функционал поиска и обновления данных.
50 000 тысяч сотрудников, у каждого есть руководитель (исключение "Big Boss"), добавлен функционал небольшого отчёта ♥
## Технические требования
- Python 3.7+
- Flask
- Flask-SQLAlchemy
- MySQL 5.6+ / PostgreSQL 10+
- Git (для управления версиями кода)
P.S.
При желании можно удалить созданный ранее db и запустить "create_db.py", будет создан новый db для анализа данных
